命令

esc模式下

光标移动

  • w -> one word
  • b <- one word
  • f+ -> 下一个字符
  • F+*
  • t+ -> 下一个字符的前一个字符
  • T+*
  • gg -> 文件头
  • 123gg 跳到123行
  • G -> 文件末尾
  • 0 -> 行头
  • $ -> 行尾
  • 星号 -> 下一个相同单词
  • 井号 -> 上一个相同单词

改动

  • . 重复上一个命令
  • o 在当前行后插入新的一行
  • O 在当前行前插入新的一行
  • yy 复制一行
  • p 粘贴,在当前位置之后 10p 粘贴10次
  • P 粘贴，在当前位置之前
  • dd 删除一行 10dd 删除10行
  • x 删除一个字符
  • dw 删除一个word
  • df+ 删除至下一个字符

  • :set nu 显示行号
