macOS command

发表于 2016-08-11

iterm 快捷键

  • command + ; 查看历史命令
  • command + shift + h 查看剪贴板历史
  • command + r 清屏
  • ctrl + r 搜索命令历史
  • ctrl + u 清楚当前行
  • ctrl + a 到行首
  • ctrl + e 到行尾
  • ctrl + w 删除光标之前的单词

mac 快捷键

  • shift + command + . 显示隐藏.文件
      • 中文输入法选词
  • 中文输入法下 shift+6 输入表情符号
  • command + `` 同组程序间切换
  • shift + command + 4 截屏，加上ctrl可以把截屏保存到剪贴板
  • command 2秒钟 显示当前应用快捷键

命令

  • history | grep xxx 查看历史命令中包含xxx的
  • htop 查看内存cpu
  • open -a Safari 命令下打开浏览器
  • pmset noidle 保持电脑不睡眠
  • du -sh 文件夹大小
  • scp -P xxx.jar root@10.10.186.43:~
  • jar -xvf xxx.jar 解压jar包
  • jps -lvm | grep callback
  • nohup java -jar xxx.jar & 后台运行
  • curl ifconfig.me 查看机器外网ip
  • linux 下： netstat –apn | grep 8080 查看8080端口的占用
  • mac 下： lsof -i tcp:8088
